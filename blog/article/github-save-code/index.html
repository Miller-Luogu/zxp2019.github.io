胶片上的代码

「转载」北极冻土里的代码，是GitHub的终极浪漫

新闻   其它   2020-07-20 16:45   2020-07-20 16:45   Text  

本文章转载自新浪科技，原文链接：https://tech.sina.com.cn/roll/2020-07-18/doc-iivhvpwx6141694.shtml。

来源：极客公园 在野

这是身处 2020 年的人类，留给未来历史学家们的礼物。

在常年冰封的 Svalbard ——斯瓦尔巴群岛，坐落着世界上最北端的小镇。这里气候极寒，数千只北极熊与三千人类居民共生。受国际斯瓦尔巴条约的监管，它是地球上最偏远，也是地缘政治最稳定的人类居住地之一。

在 GitHub 的一项异常浪漫的人类计划里，Svalbard 被选作存放现代人类文明火种的最佳位置。7 月 8 日，GitHub 将支撑当今社会运转的几乎所有开源代码，以胶片的形式储存，打包运往 Svalbard 。目前，这些胶片已经被放置在钢制容器内，密封在一个废弃煤矿中。在这个极其干燥的、-5°C 恒温的矿井中，这些代码至多可以保存一千年。

△ GitHub 代码存储地，一个位于 Svalbard 群岛上的废弃矿井|GitHub

这是身处 2020 年的人类，留给未来历史学家们的礼物。在官方声明中，GitHub 写到，‘未来的历史学家将能够从开源项目和元数据中了解我们，他们可能会认为我们这个开源的、志愿的、遵循摩尔定律的时代具有历史意义。’

末日重启基地

虽然 Svalbard 会受到气候变化的影响，但在可预见的未来，受到影响的大概率只是最外层几米的永久冻土，气候变暖不会威胁矿井的稳定。因此，这个矿井也成为保存人类文明的重要基地。

AWA（Arctic World Archive，世界数据档案馆）就在这里，除 GitHub 的代码外，AWA 已经保存了来自意大利、巴西、挪威、梵蒂冈和许多其他国家的历史和文化数据。著名的‘全球种子库’也位于此，距离 GitHub 选择的矿井仅一英里远，那里保存了世界上几乎所有农作物的种子。

△ 储存开源代码所用的胶片|GitHub

或许是静待末日重启，又或者是在平安顺遂的数百年后等待历史学家考古。被冰封在 Svalbard 的人类文明，无论是种子还是代码，对于未来的人类来说，都承载着像如今马丘比丘和吴哥窟一样的历史意义。

计算机世界疯狂迭代，被迭代掉的代码往往惨遭丢弃，或者被放进不足够稳定的存储介质中。毫不危言耸听，若不采取行动，当下社会中的数字资料，在未来几代人的时间周期内，将会消失殆尽。

GitHub 的初心来源于此。硬盘、磁带、CD、SSD，这些介质在‘保存人类文明’这件事面前，都显得不足够稳定。地理位置也是考虑因素之一，战争也好，自然灾害也罢，天灾人祸时有发生。作为当下数字社会运转的基石，如何让开源代码长期、安全地被保存下来，是 GitHub 想要成就的事。

△ 存储开源代码的胶片被打包运往 Svalbard|GitHub

胶片上的代码

由于硬件比大多数存储介质的寿命长很多，因此可能在未来某些情况下，能够正常运转的计算机硬件被保留下来，而运行在其上的软件代码无迹可寻。又或者由于网络原因，人类无法从远端服务器获取代码，此时这些被印在胶片上的代码便成为人类最后的指望。

在 GitHub 列出的被封存开源代码清单中，你会发现大量当今计算世界及网络世界的基石。Net、ruby、apache、docker、php……涵盖了操作系统、数据库、金融区块链、音视频、前后端等等几乎所有的程序智慧。

据官方文档，本次存档的档案库共含有 201 个胶卷，共计 21TB。其中包括一个含有人类可读信息和指引的‘指南胶卷’和 200 个软件存档胶卷。每个胶卷包含 65000 个单独的帧。每个胶卷开头的帧及‘指南胶卷’内的帧均包含人类可读文本和图像，胶卷的所有其它帧则由以二维码可视形式存储的 0 和 1 组成。

在未来，当人类需要读取胶卷上的代码时，只需阅读‘指南胶卷’上的内容，并按指引将二维码解压为 0 和 1 构成的二进制文件，再将二进制文件解压为一个更长但已解压的存档文件，最后将存档文件解包为其所包含的不同子文件，即可获得被保存的代码。

GitHub 透露，胶片并非该项目的最终存储介质，一种采用二氧化硅（也即石英玻璃）的存储技术正在开发过程中。石英玻璃能够提供上万年的存储寿命，并且其对电磁干扰、水和热都有很好的适应性，比胶片更为理想。目前 GitHub 已经将 6000 个世界范围内最受欢迎的项目代码以石英玻璃的形式存储，以验证新技术的可行性。

遗失的技术有的时候能够在未来帮助人类。

比如古罗马混凝土配方堪称完美，经久耐用屹立不倒，是人类历史上最耐用的建筑材料，却曾因战乱一度险些失传。又比如 20 世纪上半叶被广泛使用的抗病虫害及疟疾手法 DDT，如今又被联合国呼吁重拾。在复盘挑战者号失事的过程中，美国航天界人士认为其根源在于土星 5 号火箭第二次生产计划的取消，若用火星 5 号运载，可避免事故的发生。

与人们的普遍判断不同的是，科技与文明并非一直滚滚向前。我们很容易想象，在未来，现在的软件代码应该会被人们看作是某种古怪的、接近遗失的文明，直到某个意想不到的需求出现。将如今的技术快照存档，或许将在未来某个随机时刻帮助人类度过难关。
